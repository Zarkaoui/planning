

- PlanningAbsences v1.0.2 : demo available at http://planningabsences.free.fr/

- See changelog file

- Must read LICENCE files before doing anything.

- Need LAMP/WAMP/MAMP server (Apache+MySQL+PHP)

-> SETUP

- To Install, copy into web dir (eg /var/www/planning/) and go to url (eg http://planning/)

- Installation should start. If not, go to http://planning/install/index.php

-> PATCH / UPDATE

- For patch or update, you need to backup this files:
 /include/db.config.php
 /include/config.php
 /images/companyLogo.png (if you changed logo)

- decompress archive into Web directory, and overwrite files


If you want sources, just download and extract the ZIP


For problems and/or questions: use Tracker or Forum at the Sourceforge Project Page:
https://sourceforge.net/tracker/?group_id=514202

- PlanningAbsences v1.0.2 : demo disponible sur  http://planningabsences.free.fr/

- Les modifications apparaissent dans le fichier changelog.txt

- Veuillez lire le fichier LICENCE avant installation et utilisation.

- Necessite un serveur type LAMP/WAMP/MAMP (Apache+MySQL+PHP)

-> SETUP

- Pour l'installation, copiez le contenu de l'archive decompressee dans votre repertoire web (ex /var/www/planning/) puis
saisissez l'url correspondante dans votre navigateur (ex http://planning/)

- L'installation devrait demarrer automatiquement, sinon aller dans http://planning/install/index.php

-> PATCH / MISE A JOUR

- Lors de l'application d'un patch ou d'une mise a jour, pensez a effectuer une sauvegarde des fichiers suivants:
 /include/db.config.php
 /include/config.php
 /images/companyLogo.png (si vous avez change le logo)

- Decompresser l'archive dans votre repertoire web, et ecrasez les fichiers modifies


Si vous désirez les sources, récupérez et décompressez l'archive ZIP.


Pour les problemes et/ou questions: merci d'utiliser le Tracker sur la page du projet Sourceforge:
https://sourceforge.net/tracker/?group_id=514202
